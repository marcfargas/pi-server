# @marcfargas/pi-client

Terminal TUI client for pi-server — detachable agent sessions.

Connects to a running `pi-server` instance and renders the agent session as an interactive Ink terminal UI. Supports reconnection, extension dialogs, tool output, and thinking blocks.

## Install

```bash
npm install -g @marcfargas/pi-client
```

## Usage

```bash
pi-client --url ws://localhost:3000
```

See the [root README](../../README.md) for full documentation.
